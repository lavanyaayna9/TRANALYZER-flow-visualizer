Trace capture procedure
-----------------------
1) Start workstation
2) Start Ubuntu VM
3) Record traffic on Workstation1
    sudo tcpdump -s 0 -i br0 -w ptunnel.dmp
4) Start the ptunnel proxy on WM1
    sudo ./ptunnel
5) Start the ptunnel client on Workstation1
    sudo ./ptunnel -p 10.20.5.48 -lp 4567 -da nio.lu -dp 22
6) Connect via SSH to nio.lu through the ICMP tunnel
    ssh -p 4567 root@localhost
7) Type a few SSH commands to generate traffic
